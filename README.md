# DIGIY CARRIÈRE · Pilote Hôtellerie

Module pilote discret pour valoriser les profils issus d’une école hôtelière et de leurs stages terrain.

## Vision

**L’école forme.  
L’entreprise de stage confirme.  
DIGIY valorise.  
L’employeur choisit.**

DIGIY CARRIÈRE n’est pas une agence de placement, ni une promesse de départ.  
Le module sert à préparer, structurer et présenter des profils professionnels de manière claire, propre et partageable.

## Objectif du pilote

Créer une première base de profils hôtellerie-restauration avec :

- une fiche talent claire ;
- un métier identifié ;
- une formation suivie ;
- un stage terrain confirmé ;
- des compétences observées ;
- un niveau de français ;
- une présentation courte ;
- un résumé partageable par WhatsApp ;
- un QR ou lien profil à terme.

## Positionnement

DIGIY CARRIÈRE accompagne la valorisation professionnelle des talents.

Le module ne garantit pas :

- un emploi ;
- un contrat ;
- un visa ;
- un départ ;
- une embauche en France ou ailleurs.

Il finance et organise uniquement la préparation, la présentation et la mise en valeur du profil professionnel.

## Public visé

Pour la phase pilote :

- élèves d’école hôtelière ;
- anciens élèves ;
- stagiaires terrain ;
- profils recommandés par l’école ;
- profils confirmés par une entreprise de stage.

Le module n’est pas encore destiné à une inscription publique massive.

## Métiers ciblés au départ

- Cuisine
- Service
- Réception
- Chambre / étage
- Plonge
- Bar
- Pâtisserie
- Aide cuisine

## Fonctionnalités de la page

Le fichier `index.html` contient :

- une page autonome ;
- une balise `noindex` pour rester discret ;
- un message de protection clair ;
- une présentation du pilote ;
- un formulaire simple de fiche talent ;
- une génération de résumé professionnel ;
- une copie rapide pour WhatsApp ou échange avec un directeur / partenaire.

## Utilisation

Déposer simplement le fichier `index.html` à la racine du dépôt GitHub Pages.

Exemple :

```text
/index.html
/README.md
```

Le module peut être publié sur un sous-domaine séparé, par exemple :

```text
carriere.digiylyfe.com
```

ou gardé en test privé avant connexion à la vitrine DIGIYLYFE.

## Doctrine de protection

Phrase à conserver dans la page et les échanges :

> DIGIY CARRIÈRE prépare et valorise les profils professionnels. Ce service ne garantit ni emploi, ni contrat, ni visa, ni départ.

## Phase pilote recommandée

Démarrer avec 10 profils maximum :

- 3 profils cuisine ;
- 3 profils service ;
- 2 profils réception ;
- 2 profils chambre / étage.

Objectif : tester la réaction de l’école, des élèves, des entreprises de stage et des premiers employeurs.

## Statut

Module en phase pilote fermée.  
Non relié à la vitrine publique DIGIYLYFE pour le moment.
